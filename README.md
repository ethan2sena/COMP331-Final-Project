# COMP331-Final-Project

This project explores data quality aspects regarding a credit card fraud detection data-set and its applications specifically in data mining and machine learning. The analysis emphasizes the impact of data quality dimensions, particularly completeness and consistency, on model performance and reliability. It also investigates how sampling bias influences fraud detection outcomes. This repository includes the full workflow, from data preprocessing and exploratory analysis to model development and evaluation.

Link to the dataset used: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
